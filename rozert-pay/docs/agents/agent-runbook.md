# Операционный Чеклист Агента

Этот файл содержит краткий чеклист ежедневной работы агента.
Единый источник правил, ограничений и политик: `AGENTS.md`.

## Быстрые ссылки

- Основные правила и ограничения: `AGENTS.md`
- Детали платежного флоу: `.agents/skills/payment-transaction-workflow/SKILL.md`
- Детали стиля разработки: `.agents/skills/code-style/SKILL.md`
- Процесс интеграции платежек: `.agents/skills/payment-integration-design-first/SKILL.md`

## Чеклист задачи

1. Прочитать целевой код и связанные импорты.
2. Внести минимально достаточные изменения.
3. Сначала запустить таргетные тесты.
4. Запустить нужные статические проверки (`mypy`, `lint`, `pylint`) по критичности задачи.
5. Подготовить итоговый отчет с измененными файлами и выполненными проверками.

## Шаблон отчета

Используй этот компактный формат в финальном ответе:

```text
Измененные файлы:
- path/to/file.py

Проверки:
- <command> — успешно/ошибка

Не запущено / заблокировано:
- <command> — причина
```
